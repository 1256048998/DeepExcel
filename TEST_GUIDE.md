# DeepExcel 快速测试指南

## 🚀 三步上手

### 第一步：构建项目

#### 方式 A：Visual Studio（推荐）
1. 双击打开 `DeepExcel.sln`
2. 选择配置：`Release` + `Any CPU`
3. 按 `F6` 生成解决方案
4. 确认"生成成功"

#### 方式 B：命令行
```powershell
cd DeepExcel
dotnet build src/DeepExcel.AddIn/DeepExcel.AddIn.csproj -c Release
```

### 第二步：注册插件

1. **以管理员身份**打开 PowerShell
2. 运行：
```powershell
cd DeepExcel
.\scripts\register.ps1
```
3. 看到"注册成功"提示后继续

### 第三步：开始测试

1. 打开 Excel（新建空白工作簿）
2. 功能区找到 **DeepExcel** 选项卡
3. 点击 **"打开面板"** 按钮
4. 在面板中输入需求，开始测试！

## 🧪 测试用例

### ✅ 基础功能测试

#### 1. 公式生成
**准备**：在 A1:A10 输入任意数字
**输入**：`帮我计算这些数字的总和，放在A11`
**预期**：
- AI 调用 `write_formula` 工具
- A11 出现 =SUM(A1:A10) 公式
- 结果正确显示

#### 2. 数据清洗
**准备**：打开 `test-data/销售数据.csv`
**输入**：`帮我清洗销售额这列，去掉前后空格，把文本转成数字`
**预期**：
- AI 调用 `clean_data` 工具
- 数据被正确清洗

#### 3. 图表创建
**准备**：选中 A1:B10（日期和销售额）
**输入**：`用这些数据创建一个折线图，展示销售趋势`
**预期**：
- AI 调用 `create_chart` 工具
- 生成折线图，标题正确

#### 4. 透视表创建
**准备**：选中整个数据区域（含表头）
**输入**：`创建数据透视表，按地区汇总销售额，放在新工作表`
**预期**：
- AI 调用 `create_pivot_table` 工具
- 新工作表中出现透视表

#### 5. VBA 执行
**输入**：`用VBA给当前选中的单元格添加黄色背景色`
**预期**：
- 弹出二次确认对话框（高风险操作）
- 确认后执行 VBA
- 单元格背景变黄

#### 6. Python 执行
**输入**：`用Python计算A列每个数的平方，结果写入B列`
**预期**：
- AI 调用 `execute_python` 工具
- B 列写入计算结果

## 🔍 常见问题排查

### 问题：Excel 里看不到 DeepExcel 选项卡
**检查**：
1. 文件 → 选项 → 加载项
2. 底部"管理"选 **COM 加载项** → 转到
3. 看列表里有没有 `DeepExcel.AddIn`
   - ❌ 没有 → 重新运行 `register.ps1`
   - ⚠️ 未勾选 → 勾选它
   - ⚠️ 在"禁用项目"里 → 启用它

### 问题：面板打开是空白的
**检查**：
1. 确认 DLL 同目录下有 `WebViewAssets` 文件夹
2. 确认文件夹里有 `index.html`
3. 检查是否安装了 WebView2 Runtime

### 问题：AI 没有回复
**检查**：
1. 配置了正确的 API Key 吗？
2. 网络能访问模型接口吗？
3. 查看日志文件：
   ```
   %AppData%\DeepExcel\logs\
   ```

### 问题：VBA 执行失败
**检查**：
1. Excel → 文件 → 选项 → 信任中心 → 信任中心设置 → 宏设置
2. 选择"启用所有宏"（测试用完改回来）
