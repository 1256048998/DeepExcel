# DeepExcel 测试用例目录

本目录包含DeepExcel插件的测试用例，用于验证插件各项功能。

## 目录结构

```
test-cases/
├── README.md                    # 本文件
├── 测试用例总览.md               # 所有用例的汇总清单
├── 公式问题测试用例.md           # 公式生成功能测试
├── VBA问题测试用例.md            # VBA执行功能测试
├── 数据清洗测试用例.md           # 数据清洗功能测试
├── 图表透视表测试用例.md         # 图表和透视表功能测试
└── Python执行测试用例.md         # Python执行功能测试
```

## 测试用例说明

每个测试用例包含：
- **用例ID**：唯一标识符（如F-001、V-010）
- **用户需求**：自然语言描述的需求
- **输入数据**：测试所需的数据
- **预期结果**：期望的输出或行为
- **验证点**：判断测试是否通过的标准

## 使用方法

### 对于测试人员
1. 阅读 `测试用例总览.md` 了解整体测试范围
2. 按优先级执行P0核心用例
3. 记录测试结果

### 对于Code X
1. 参考各测试用例文件中的需求描述
2. 模拟用户输入
3. 验证插件的响应和执行结果

## 测试优先级

- **P0**：核心功能，必须通过（公式生成、VBA执行、数据清洗）
- **P1**：重要功能，应该通过（图表、透视表）
- **P2**：增强功能，可以测试（Python执行、高级分析）
