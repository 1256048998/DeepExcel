# DeepExcel 图表与透视表功能测试用例

> 来源：Excel用户常见图表和透视表问题整理
> 日期：2026-06-29

---

## 一、基础图表创建

### 测试用例 C-001：折线图
- **用户需求**：`用A列的日期和B列的销售额创建一个折线图，展示销售趋势`
- **输入数据**：
  - A列：日期（2026-01-01 到 2026-01-30）
  - B列：对应销售额
- **预期结果**：
  - 生成折线图
  - X轴为日期，Y轴为销售额
  - 图表标题为"销售趋势"
- **验证点**：图表类型正确、数据源正确、标题有意义

### 测试用例 C-002：柱状图
- **用户需求**：`用A列的产品名称和B列的销量创建柱状图`
- **输入数据**：
  - A列：产品名称（手机、电脑、平板...）
  - B列：销量
- **预期结果**：
  - 生成柱状图
  - X轴为产品，Y轴为销量
- **验证点**：类别标签正确显示

### 测试用例 C-003：饼图
- **用户需求**：`用A列的部门和B列的人数创建饼图，展示人员分布`
- **输入数据**：
  - A列：部门名称
  - B列：人数
- **预期结果**：
  - 生成饼图
  - 显示百分比标签
- **验证点**：百分比总和为100%

### 测试用例 C-004：散点图
- **用户需求**：`用A列的广告投入和B列的销售额做散点图，分析相关性`
- **输入数据**：
  - A列：广告投入金额
  - B列：销售额
- **预期结果**：
  - 生成散点图
  - 每个点对应一行数据
- **验证点**：X/Y坐标正确

## 三、组合图表

### 测试用例 C-020：柱状图+折线图
- **用户需求**：`用柱状图展示销量，折线图展示增长率，做成组合图`
- **输入数据**：
  - A列：月份
  - B列：销量（柱状图）
  - C列：增长率（折线图，次坐标轴）
- **预期结果**：
  - 柱状图在主坐标轴
  - 折线图在次坐标轴
- **验证点**：双坐标轴正确、图例区分

### 测试用例 C-021：多系列柱状图
- **用户需求**：`对比各产品在Q1和Q2的销量`
- **输入数据**：
  - A列：产品名称
  - B列：Q1销量
  - C列：Q2销量
- **预期结果**：每个产品有两根柱子，颜色不同
- **验证点**：系列正确区分

## 四、数据透视表

### 测试用例 P-001：基础透视表
- **用户需求**：`创建透视表，按地区汇总销售额`
- **输入数据**：
  - A列：产品
  - B列：地区
  - C列：销售额
- **预期结果**：
  - 行标签：地区
  - 值：销售额求和
- **验证点**：数值正确、自动求和

### 测试用例 P-002：多维度透视
- **用户需求**：`按地区和产品类别交叉汇总销售额`
- **预期结果**：
  - 行：地区
  - 列：产品类别
  - 值：销售额
- **验证点**：交叉表正确

### 测试用例 P-003：透视表筛选
- **用户需求**：`在透视表中添加筛选器，可以按年份筛选`
- **预期结果**：筛选器可用、数据正确筛选
- **验证点**：筛选结果正确

### 测试用例 P-004：透视表计算字段
- **用户需求**：`在透视表中添加利润率字段（销售额/成本）`
- **预期结果**：新字段正确计算
- **验证点**：公式正确、数值准确

## 注意事项

1. **图表位置**：默认在当前工作表创建，用户可指定新工作表
2. **数据源验证**：创建前检查数据是否有效
3. **图表尺寸**：默认适中，用户可调整
4. **中文支持**：标题、标签支持中文字符
